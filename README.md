# AI导航

一个静态网页AI导航，收集和整理优质的人工智能工具与资源。

## 功能特性

- 🎨 现代化响应式设计
- 📱 支持移动端和桌面端
- 🔗 精选AI工具链接
- 🚀 快速访问

## 已收录工具

- **OpenClaw** - 跨平台消息应用网关，支持WhatsApp、Telegram、Discord和iMessage，连接AI代理

## 如何使用

1. 克隆或下载本仓库
2. 打开 `index.html` 文件即可访问导航页面
3. 点击相应工具的链接进入官方网站

## 如何贡献

欢迎添加更多优质的AI工具和资源！只需编辑 `index.html` 文件，按照现有格式添加新的工具卡片即可。

## 部署到GitHub Pages

1. 在GitHub上创建一个新仓库
2. 将本项目的文件推送到仓库
3. 在仓库设置中启用GitHub Pages，选择 `main` 分支作为源
4. 等待几分钟后，即可通过GitHub Pages提供的URL访问导航页面
